## 2. ユーザーストーリー

### プライマリーユーザー

PC ゲーマー（Windows）。特に以下の課題を持つ人：

- ゲームのたびに「OneDrive 止めて / 電源プラン変えて / DNS 変えて」を繰り返している
- PC の設定を変えすぎて元に戻せなくなった経験がある
- ハードウェアの状態（温度・VRAM 等）が気になる

### ストーリーマップ

```
[発見]  アプリ起動 → 初回ウィザード(APIキー設定) → ホーム画面
[最適化] プロセス停止 → 電源プラン変更 → Windows設定 → ネットワーク設定 → スコア確認
[ゲーム] ゲームライブラリ → プロファイル選択 → 起動 → セッション記録
[復元]  ロールバックセンター → セッション選択 → 一括復元
[監視]  トレイ常駐 → 自動最適化 ON → ゲーム検知 → 自動プロファイル適用
[更新]  アップデート画面 → ダウンロード → インストール → 自動再起動
```

## 5. 既知の制限事項

| 項目 | 詳細 |
|---|---|
| GPU 制御 | nvidia-smi 依存のため AMD/Intel GPU は非対応 |
| FPS 計測 | OS レベルの推定のため精度は低い |
| ゲーム起動検知 | exe プロセス名ポーリングのため遅延あり（数秒） |
| Epic/GOG/Xbox 検出 | FF=false（次期対応予定） |
| HAGS/Defender 除外 | FF=false（次期対応予定） |
| ベンチマーク履歴 | FF=false（次期対応予定） |
| macOS/Linux | 非対応（Windows 専用） |

## 6. セキュリティ要件

- API キーは OS の keyring（認証情報マネージャー）に保存する
- `CREATE_NO_WINDOW` フラグで外部プロセスのコンソール非表示
- 自動更新は署名付きバイナリのみ受け付ける（minisign）
- レジストリ操作はリスク分類（safe/caution/advanced）を表示してから実行
