# Pre-Deploy Test Checklist — Gaming PC Optimizer v1.0.0

> 環境: Windows 10/11、NVIDIA GPU 推奨。各項目を実機で 1 回ずつ確認し ✅ を付ける。

---

## 1. 起動・終了

- [ ] `cargo tauri dev` または配布インストーラからアプリを起動 → メイン画面が表示される
- [ ] ウィンドウ右上の ✕ ボタン → ウィンドウが閉じ、タスクバー通知領域にトレイアイコンが残る
- [ ] トレイアイコン左クリック → ウィンドウが表示される / 再度クリックで非表示に切り替わる
- [ ] トレイ右クリック → 「ダッシュボードを開く」でウィンドウが前面に出る
- [ ] トレイ右クリック → 「終了」でプロセスが完全に終了する（タスクマネージャーで確認）

---

## 2. ダッシュボード

- [ ] CPU / メモリ / GPU / ネットワーク の MiniStat が数値表示される
- [ ] ヘルスリングが 0〜100 のスコアを表示する
- [ ] 「ワンクリック最適化」ボタンを押すと処理が走り、完了ステータスが表示される

---

## 3. GameMode（最適化）

- [ ] 「スキャン」ボタンでバックグラウンドプロセスが一覧表示される
- [ ] プロセス行にアイコン（EXE アイコン or 頭文字アバター）が表示される
- [ ] 「ワンクリック最適化」で不要プロセスが停止し、結果ログが出る
- [ ] 「元に戻す」で停止したプロセスの復元処理が走る

---

## 4. My Games（ゲームライブラリ）

- [ ] 「Steam スキャン」でインストール済みゲームが自動追加される
- [ ] Steam ゲームのカードに Steam ヘッダー画像が表示される（取得できない場合はグラデーション プレースホルダー）
- [ ] 「＋追加」から手動でゲームを登録できる
- [ ] ゲームカードの「最適化して起動」ボタンでゲームが起動する
- [ ] Steam 以外（EXE 直接）のゲームも登録・起動できる

---

## 5. ハードウェア

- [ ] GPU 情報（名前・VRAM・温度・使用率）が表示される
- [ ] GPU ベンダーロゴ（NVIDIA 緑 / AMD 赤 / Intel 青）が表示される
- [ ] CPU 情報・マザーボードブランドアイコンが表示される
- [ ] 電力モード（パフォーマンス / バランス / バッテリー）の切り替えが動作する
- [ ] AI モード推奨ボタンを押すと推奨モードが表示される

---

## 6. アップデート

- [ ] 起動時に winget でアプリ一覧が自動スキャンされる
- [ ] アップデート可能なアプリにパブリッシャー別ブランドカラーのアバターが表示される
- [ ] ドライバー一覧でNVIDIA/AMD/Intel ドライバーにブランドアイコンが表示される
- [ ] 「AI 優先度分析」で重要 / 推奨 / 任意のバッジが付く
- [ ] チェックボックスで選択して「〇件を更新」ボタンで winget アップグレードが走る

---

## 7. ストレージ管理

- [ ] 「スキャン」でカテゴリ別ファイルサイズ一覧が表示される（カテゴリ別アイコン付き）
- [ ] 「AIに推奨してもらう」で削除OK / スキップ バッジが付く
- [ ] チェックボックスで選択し「削除」を実行 → 解放サイズが表示される
- [ ] アクセス不可カテゴリはグレーアウトされ削除されない

---

## 8. ネットワーク最適化

- [ ] ネットワークアダプター情報が取得される
- [ ] DNS プリセット（Google / Cloudflare / OpenDNS）への切り替えが動作する
- [ ] 「元に戻す」で DNS が DHCP に戻る
- [ ] Ping テストが実行され結果（ms）が表示される

---

## 9. WindowsSettings

- [ ] 現在の設定値（視覚効果 / Game DVR / 透過 / アニメ等）が読み込まれる
- [ ] 「ゲーミング設定を適用」で変更が走り、設定復元用バックアップが作成される
- [ ] 「元に戻す」でバックアップから復元される

---

## 10. プロファイル管理

- [ ] プロファイル一覧が表示される
- [ ] 新規プロファイルを作成・保存できる
- [ ] プロファイルを適用するとゲーム起動時に設定が反映される（ウォッチャー連動）
- [ ] プロファイルを削除できる

---

## 11. 設定（AIキー）

- [ ] APIキーが未設定の状態で AI ボタン（各ページ）を押すと「APIキーが設定されていません」というエラーが UI に表示される（クラッシュしない）
- [ ] 設定画面で Anthropic APIキーを入力・保存し、AI 機能が動作する
- [ ] APIキーを削除して再度 AI ボタン → エラーが出る

---

## 12. 自動最適化（ウォッチャー）

- [ ] トレイメニュー「自動最適化」チェック → ゲーム起動検知で自動プロファイル適用が動作する
- [ ] 「すべて元に戻す」でウォッチャーが設定を元に戻す

---

## ビルド確認メモ

```
npm run build          # TypeScript コンパイル + Vite バンドル
cargo tauri build      # Rust コンパイル + Windows インストーラ生成
# 出力先: src-tauri/target/release/bundle/
```

- [ ] `npm run build` がエラーなく完了する
- [ ] `cargo tauri build` がエラーなく完了する
- [ ] `src-tauri/target/release/bundle/msi/` に `.msi` が生成される
- [ ] `.msi` インストーラを実行して正常にインストール・起動できる
- [ ] アンインストールが正常に動作する
